# flutter-gemini-ai-app
Basic Text to Text AI with Gemini AI in Flutter

1.Open the Flutter project in Visual Studio or Android Studio (tested VS)

2. Run flutter pub get command in terminal
  
3. Go to the apikey.dart file and enter the your api key.

Dart SDK version must be 3.2.0 or above

An API key is required to use the Gemini API. If you don't have a key, create one in Google AI Studio.

https://aistudio.google.com/app/apikey
